# **App Name**: Qrlive-Menu

## Core Features:

- Two Column Layout: Displays content in a clear two-column layout.
- Color Scheme Assignment: Assigns unique color schemes to each column for easy differentiation.
- Content Area: Define discrete scrollable areas within each column.

## Style Guidelines:

- Primary Color: #0B2447 for a calming, yet distinct base.
- Background Color: White, to give maximum contrast and legibility.
- Accent Color: Light grey as an analogous hue, provides a visual distinction while maintaining aesthetic harmony.
- Text Color: Black for readability
- Font: 'Inter' (sans-serif) for a clean and modern reading experience; for both headings and body text.
- Use a balanced layout with clear separation between columns to enhance readability.
